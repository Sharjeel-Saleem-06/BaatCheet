### Chat Endpoints

| Method | Endpoint | Description |
|:------:|:---------|:------------|
| `POST` | `/api/v1/chat/completions` | Send chat message (SSE streaming) |
| `POST` | `/api/v1/chat/regenerate` | Regenerate last response |
| `GET` | `/api/v1/chat/models` | List available AI models |
| `GET` | `/api/v1/chat/providers/health` | Provider status check |

### Vision & OCR Endpoints

| Method | Endpoint | Description |
|:------:|:---------|:------------|
| `POST` | `/api/v1/chat/vision/analyze` | Analyze image with AI |
| `POST` | `/api/v1/chat/ocr/extract` | Extract text from image |
| `POST` | `/api/v1/chat/ocr/process` | OCR + AI processing |

### Voice Endpoints

| Method | Endpoint | Description |
|:------:|:---------|:------------|
| `POST` | `/api/v1/chat/tts` | Text-to-speech |
| `POST` | `/api/v1/chat/voice/transcribe` | Speech-to-text |

### Project Endpoints

| Method | Endpoint | Description |
|:------:|:---------|:------------|
